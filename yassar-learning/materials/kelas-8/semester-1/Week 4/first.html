It's All About Logic: "If..., Then..."

🌧️

IF it is raining, THEN I will bring an umbrella.

💯

IF a student's score is >= 75, THEN they have "Passed".

The IF function does the exact same thing inside your spreadsheet!

The Anatomy of an IF Function

=IF(logical_test, value_if_true, value_if_false)
1. Logical Test: The condition to check. (e.g., Is the score in C2 greater than or equal to 75? -> C2>=75)
2. Value if True: What to show if the condition is met. (e.g., "Passed")
3. Value if False: What to show if the condition is NOT met. (e.g., "Failed")
